Présentation
------------

L'Assistant RAG Multi-Modèles est une application Streamlit avancée conçue pour charger, traiter et interroger des documents en utilisant différents modèles de langage (LLM) avec une approche de Retrieval-Augmented Generation (RAG).

Fonctionnalités Principales
--------------------------

- Chargement de documents multi-formats (PDF, TXT, DOCX)
- Support de plusieurs modèles LLM
- Vectorisation dynamique des documents
- Interface utilisateur intuitive Streamlit
